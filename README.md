# 3.7Inch-Eink
A small but versatile handheld ebook

这是一个3.7英寸的 手持电子书

它带有

·ESP32 WROOM32UE

·MAX98357音频功放

·RX8025T时钟芯片

·SHT30温湿度传感器

·1511扬声器

·1000mAh 3.7Wh的锂电池

·TP4055充电模块

·采用了OKRA 的3.7寸黑白墨水屏作为显示设备

·三个凯华微动按键作为输入设备

·时钟模式下至少能够待机40天
